Express
=======

# What is it?
  * A node.js framework for building web applications

# What's awesome about it?
  * CLI easily bootstraps a new site
  * Can easily serve up webpages or APIs
  * Simple syntax for:
    * Serving html
    * Serving JSON
    * Defining Routes
    * Defining Middleware
  * Very pluggable
    * View Engines
    * JS/CSS Preprocessors
    * Middleware
  * Popular
    * Easy to find help/documentation
